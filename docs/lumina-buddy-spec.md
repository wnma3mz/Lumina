# Lumina Buddy — Web 终端 ASCII 宠物设计规范 (v3.0)

本文件定义了 Lumina Buddy 的高级演化逻辑：让宠物成为用户当日行为的「镜像」。

## 1. 核心定位
Lumina Buddy 不仅是 AI 伴侣，更是用户当日数字活动的视觉总结。它会根据「今日回顾」的数据分布改变外形、性格和台词。

## 2. 宠物系统设计

### 2.1 基础属性与性格 (已实装)
- 基于用户哈希的五维属性（持久化）。

### 2.2 成长与等级 (已实装)
- XP 累计与 Lv.1-Lv.99 等级系统。

### 2.3 今日演化系统 (v3.0 核心)
- **Vibe Analysis**: 每天读取「数据源脉动」比例。
- **Dynamic Accents**: 
  - `Code-Heavy`: 增加 `[ ]` 护目镜装饰。
  - `Web-Heavy`: 增加 `( )` 眼镜装饰。
  - `Plan-Heavy`: 增加 `~` 慵懒装饰。
- **Contextual Greetings**: 
  - 根据当日采集到的最高活跃源，定制首页第一句台词。
